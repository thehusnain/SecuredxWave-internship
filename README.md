# SecuredxWave Internship Portfolio

This repository contains structured daily deliverables from a summer internship, organized by week and day.

## Repository Structure

- `offer-letter/` - Internship offer letter
- `internship-schedule/` - Internship schedule documents
- `week-01/` - Week 01 daily work (reports, assets, and task materials)

Each day folder follows a consistent layout:

- `report/` - Main daily report (`day-XX-report.docx` + `day-XX-report.pdf`)
- `assets/` - Supporting images/screenshots (when available)
- `task-details/` - Lab/task source materials (when available)
